PROFINISH REWARDS - GITHUB PAGES READY SITE

Upload these files to your GitHub repository:

1. index.html
2. the full assets folder

Do not upload this ZIP directly to GitHub. Unzip it first.

Your GitHub repository should show index.html at the top level.
The assets folder should also be at the top level.

Then go to GitHub:
Settings > Pages > Build and deployment
Source: Deploy from a branch
Branch: main
Folder: /root
Save.

Use the regular GitHub Pages link first. Add the custom domain later after the site is live.
